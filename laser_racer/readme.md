Super super edgey. This example uses the ES6 [Set](https://developer.mozilla.org/en-US/docs/JavaScript/Reference/Global_Objects/Set) API and probably won't work in your browser.

# Setup

```bash
brew install node
npm install -d
node app
```

Enjoy!
